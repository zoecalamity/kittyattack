## Description
Kitty attack is a space invaders type shooter game. The player is a space man who has to shoot evil kitty aliens. When the player shoots a kitty the kitty dies, and when the kitty collides with the player, the player dies. Points are recieved when the player shoots a kitty. 

The data folder in the src folder contains the images for the game. A player image, a kitty image, and a fish image. These images make up the player, the invader, and the bullet, respectively. I drew these assets using procreate. 

I used "Building Space Invaders Using PyGame - Python" by Geeks for Geeks as reference for this project (https://www.geeksforgeeks.org/python building-space-invaders-using-pygame-python/).
