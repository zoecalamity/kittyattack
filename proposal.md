


# Title
Kitty Attack 

## Repository
https://github.com/zoecalamity/kittyattack.git

## Description
Kitty attack is a space invaders type shooter game. It is relevant to digital media arts as it is a visual artwork. 

## Features
- Player Shoots Enemies 
	- Collisons will be defined and enemies will die when bullet collides with enemy. 
- Player dies
	- Collisions will be defined and player will die when enemy collides with player.
-  Player wins 
	- Player will win when defined number of enemies is dead. 

## Challenges
- I need to learn how to define collisions. 
- I need to learn how to move the player character and how to shoot. 
- I need to learn how to stop the game at a win point. 

## Outcomes
Ideal Outcome:
- The ideal is to have a game where the player shoots a certain number of enemies and wins the game, and dies if collided with. 

Minimal Viable Outcome:
- The bare essential is for the player to shoot an enemy and die if the enemy touches the player.

- Week N (Final)
